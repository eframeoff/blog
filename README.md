## Блог

### Учебное приложение по видеоурокам (2020)

- Клиент: localhost:3000
- Сервер:  localhost:3001
- БД - MongoDB
- Javascript, JQuery, Ejs, Sass, Node.js (< 12), Express
- Сборщик - Gulp

### Как запускать? Установка пакетов

`npm i`

### Линтинг

`npm run lint`

### Development

`npm run dev`
