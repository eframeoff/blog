## Блог

### Учебное приложение по видеоурокам (2020)

Клиент крутится на localhost:3000
Сервер крутится на localhost:3001
БД - MongoDB
На фронте - js, jquery, ejs, sass
На бэке - node.js (< 12), express
Сборщик - gulp

### Как запускать? Установка пакетов

`npm i`

### Линтинг

`npm run lint`

### Development

`npm run dev`
