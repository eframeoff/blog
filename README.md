# Блог

## Учебное приложение по видеоурокам

### Технологии
- Frontend: Javascript, JQuery, Ejs, Sass (localhost:3000)
- Backend: Node.js (< v12), Express, MongoDB (localhost:3001)
- Сборщик - Gulp

### Как запускать? Установка пакетов

`npm i`

### Линтинг

`npm run lint`

### Development

`npm run dev`
